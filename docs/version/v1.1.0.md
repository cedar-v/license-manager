# Release v1.1.0

**发布日期**：2026-01-17

---

## ✨ 新增功能

### 🔐 客户用户管理系统
- **手机号注册登录**：支持手机号注册和登录，完全独立于管理员用户体系
- **用户角色管理**：支持企业管理员和普通成员角色
- **用户资料管理**：支持头像、语言偏好设置等个性化配置
- **安全认证隔离**：采用路由分组隔离，管理员和客户用户完全分离

### 💳 产品套餐体系
- **三种套餐类型**：试用版、基础版、专业版，支持不同的功能限制和许可数量
- **阶梯定价折扣**：根据许可数量提供折扣
- **自动授权码生成**：购买后自动生成对应的授权码，支持批量许可
- **订单管理系统**：完整的订单创建、查询和管理流程

### 🔑 授权码增强功能
- **离线激活支持**：授权码自包含配置信息，支持无网络环境激活
- **产品激活码**：提供可离线解析的加密授权码格式
- **授权码分享**：用户可将其授权码的部分激活次数分享给其他用户
- **RSA数字签名**：确保授权码的安全性和防篡改

### 📱 设备管理
- **设备列表查看**：客户用户可查看所有激活授权码的设备信息
- **设备解绑**：支持解绑设备，释放授权码许可供其他设备使用
- **在线状态监控**：实时显示设备在线状态和最后心跳时间
- **设备信息展示**：显示设备名称、操作系统、硬件信息等详情

### 💰 支付宝支付集成
- **在线支付支持**：集成支付宝官方SDK，支持产品套餐购买
- **支付状态管理**：完整的支付流程和状态跟踪
- **安全支付验证**：支持支付回调验证和金额校验
- **多环境配置**：支持沙箱和生产环境切换

### 📨 短信验证码服务
- **阿里云短信集成**：基于阿里云短信服务，支持验证码发送
- **多种验证码场景**：注册验证、登录验证、密码重置等
- **频率控制**：防止短信轰炸，限制发送频率
- **验证码缓存**：Redis缓存验证码，支持过期和验证

### 🗄️ 缓存模块重构
- **多缓存实现**：支持内存缓存和Redis缓存两种方式
- **Clean Architecture**：基于Clean Architecture设计，提供统一缓存接口
- **序列化支持**：自动处理JSON序列化和反序列化
- **监控统计**：提供缓存命中率等性能监控指标

## 🐛 修复问题

- **授权码格式统一**：修复授权码生成格式不一致问题
- **支付流程优化**：优化支付流程的用户体验和错误处理
- **缓存稳定性**：提升缓存模块的稳定性和错误处理能力
