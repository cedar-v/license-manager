# Release v1.0.1

**发布日期**：2025-11-22

---

## ✨ 新增功能

- **数据库初始化重试机制**：提升系统在启动过程中的稳定性与容错能力  
- **许可证管理功能**：支持许可证的撤销与下载操作，增强管理灵活性  
- **语言切换功能**：新增语言切换入口，优化语言选择界面，提升国际化体验  
- **官方文档**：更新官方文档，增加许可证结构与验证（客户端要点）说明，商业价值说明等。

## 🛠 优化与改进

- **许可证信息展示**：更新国际化文本，优化展示逻辑，提升可读性  
- **导航与菜单栏**：完善导航栏与菜单栏结构，暂时禁用搜索与通知按钮，提升整体交互体验  
- **授权基础信息**：优化剩余天数展示逻辑，提升用户体验
